# 正三角形
### Shelter Zone Newbie Online Judge S1 Q1

## 難度：🔥🔥

## 題目
下方會給你一個由字串組成的正三角形，請你將它放入程式，並且求出這個正三角形的面積。  
```
triangle = "    ^     \n\
               / \    \n\
              /   \   \n\
             /     \  \n\
            /_______\ \n";
```

## 輸入
無

## 輸出
一浮點數，為正三角形的面積。

## 範例輸入

```
無
```
## 範例輸出
```
14.433757
```
