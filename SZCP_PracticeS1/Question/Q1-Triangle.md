# 正三角形

> ShelterZone Code Contest S1 PracticeContest Q1<br>
> 題目作者: iblackcat#4119

## 題目
下方會給你一個由字串組成的正三角形，請你將它放入程式，並且求出這個正三角形的面積。  

```
triangle = "    ^     \n\
               / \    \n\
              /   \   \n\
             /     \  \n\
            /_______\ \n";
```

---

## 輸入
無

## 輸出
一浮點數![](https://latex.codecogs.com/svg.latex?f)，為正三角形的面積。

---

## 範例輸入
```
無
```
## 範例輸出
```
14.433757
```
