# 2^N再2^C

> ShelterZone Code Contest S1 PracticeContest Q5<br>
> 題目作者: JiKuai#2121

## 題目

給定![](https://latex.codecogs.com/svg.latex?X,N)兩整數，<br>
請問是否存在非負整數![](https://latex.codecogs.com/svg.latex?C)滿足![](https://latex.codecogs.com/svg.latex?2^{N+C}=X)？

---

## 輸入

輸入第一行有一個整數![](https://latex.codecogs.com/svg.latex?T%20(1%20\le%20T%20\le%20100))，代表接下來有T筆測資，<br>
每筆測資只有一行，有兩個整數![](https://latex.codecogs.com/svg.latex?X,%20N%20(1%20\le%20X%20%3C%202^{31},%201%20\le%20N%20\le%2030))，意義已在題目敘述。

## 輸出

如果存在，輸出YES, 否則輸出NO

---

## 範例輸入

```
5
8 2
8 3
8 4
1 3
5 2
```

## 範例輸出

```
YES
YES
NO
NO
NO
```
